## Garments ERP System (Desktop Application)

A Java Swing–based desktop application developed using NetBeans to manage core operations of a garments business. The system provides user authentication and a dashboard-driven interface to handle employees, buyer orders, production, and inventory.

### Key Features
- User Registration and Login
- Dashboard-based navigation
- Employee Management
- Buyer Order Management
- Production Management
- Inventory Management
- Secure MySQL database integration using JDBC

### Technologies
- Java (Swing)
- NetBeans IDE
- MySQL
- JDBC

### Project Structure
- ui – Application forms (Login, Register, Dashboard, Modules)
- model – Data models (User, Employee)
- dao – Database access layer
- util – Database connection utility

### How to Run
1. Open the project in NetBeans
2. Configure the MySQL database connection
3. Run the application from LogInForm
